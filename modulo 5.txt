# 📋 MAXIQUEEN OS

# OPERACIÓN BASE — MÓDULO 1

## Contenido Creativo

---

# 1️⃣ IDENTIDAD DEL MÓDULO

**Objetivo Estratégico:**
Conectar emocionalmente y construir comunidad mediante narrativa estructurada.

**Resultado Medible Esperado (30 días):**

* 12–20 piezas publicadas
* 1 línea narrativa definida
* 1 formato repetible validado
* 1 micro-comunidad activa (comentarios / respuestas / DMs)

---

# 2️⃣ ESTRUCTURA OPERATIVA DEL MÓDULO

Modelo estándar replicable:

```
ENTRADA → DISEÑO → PRODUCCIÓN → PUBLICACIÓN → ANÁLISIS → OPTIMIZACIÓN
```

Este flujo se repetirá en todos los módulos.

---

# 3️⃣ FASE 1 — ENTRADA (Diagnóstico Creativo)

Checklist obligatorio:

☐ Definir nicho real
☐ Definir dolor principal del público
☐ Definir emoción dominante (miedo, frustración, ambición, culpa, esperanza)
☐ Definir mensaje central
☐ Definir tono (directo, provocador, reflexivo, técnico)

Salida de esta fase:
→ Documento “Mapa Narrativo Base”

---

# 4️⃣ FASE 2 — DISEÑO (Arquitectura Narrativa)

Aquí se construye la estructura madre.

## 4.1 Línea Narrativa Principal

Ejemplo:

* Reconstrucción
* Caos → Orden
* Evolución
* Autoridad técnica

## 4.2 Tipos de Contenido (Matriz 3x3)

| Tipo                 | Objetivo       | Frecuencia |
| -------------------- | -------------- | ---------- |
| Historia personal    | Conexión       | 1–2/sem    |
| Contenido educativo  | Autoridad      | 2–3/sem    |
| Opinión estructurada | Diferenciación | 1–2/sem    |

Salida:
→ Calendario base 30 días

---

# 5️⃣ FASE 3 — PRODUCCIÓN

Sistema de creación:

### Fórmula estándar de guion corto:

1. Hook fuerte (3–5 segundos)
2. Problema claro
3. Desarrollo simple
4. Cierre con tensión o reflexión
5. CTA ligero

Checklist producción:

☐ 3 hooks probados
☐ 1 versión corta (30–45s)
☐ 1 versión extendida (60–90s)
☐ Subtítulos listos
☐ Miniatura coherente

Salida:
→ 4–6 piezas listas por semana

---

# 6️⃣ FASE 4 — PUBLICACIÓN

Reglas operativas:

☐ Horarios definidos
☐ Copy corto complementario
☐ CTA claro
☐ Publicación cruzada (si aplica)
☐ Respuesta activa a comentarios (primeros 60 min)

Salida:
→ Distribución activa

---

# 7️⃣ FASE 5 — ANÁLISIS

Indicadores clave:

* Retención
* Comentarios
* Guardados
* Compartidos
* Clics

No se mide likes como métrica principal.

Checklist:

☐ Identificar mejor hook
☐ Identificar mejor formato
☐ Identificar mejor emoción
☐ Detectar puntos de caída

Salida:
→ Ajustes concretos

---

# 8️⃣ FASE 6 — OPTIMIZACIÓN

Sistema de mejora continua:

* Repetir lo que funciona
* Eliminar lo que no conecta
* Ajustar tono si hay fricción
* Evolucionar narrativa cada 30 días

Salida:
→ Versión 2.0 del contenido

---

# 📆 FLUJO OPERATIVO 30 DÍAS

Semana 1 → Diagnóstico + Diseño
Semana 2 → Producción base
Semana 3 → Producción + Optimización inicial
Semana 4 → Ajuste estratégico + repetición de formato ganador

---

# 📂 ENTREGABLES FORMALES DEL MÓDULO

Al cerrar el módulo, el cliente debe tener:

✔ Línea narrativa definida
✔ 1 formato validado
✔ 12–20 piezas publicadas
✔ Métricas analizadas
✔ Sistema repetible

---

# 🧠 FORMATO REPLICABLE PARA LOS OTROS 9 MÓDULOS

Todos usarán esta misma estructura:

1. Identidad
2. Objetivo medible
3. Flujo operativo
4. Checklists por fase
5. Indicadores clave
6. Entregables finales

Así el sistema se mantiene coherente.
